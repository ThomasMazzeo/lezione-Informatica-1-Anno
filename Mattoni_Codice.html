Prima di scrivere in C++, dobbiamo capire cosa stiamo manipolando. Scopriamo la memoria, la matematica del PC e come parlargli.

La Variabile

Il computer non ha memoria di per sé. Per fargli ricordare qualcosa (il punteggio, il tuo nome), dobbiamo creare una Scatola nella RAM.

int
15
eta_utente

int (Il Tipo)

Che forma ha la scatola? Ci vanno solo numeri o solo testo? Dobbiamo dirlo subito.

eta_utente (Il Nome)

Un'etichetta univoca appiccicata sulla scatola per ritrovarla quando ci serve nel programma.

= 15; (L'Assegnazione)

Mettiamo fisicamente il dato 15 dentro la scatola e chiudiamo l'istruzione col punto e virgola.

int eta_utente = 15;

I Tipi di Dato Principali

int

Numeri Interi. Senza virgola, positivi o negativi.

int vite = 3;

float / double

Numeri decimali (con la virgola, che in C++ è il punto).

float prezzo = 19.99;

string

Una catena di caratteri (Testo). Va sempre tra doppi apici.

string nome = "Mario";

bool (Booleano)

La logica binaria pura. Può contenere solo due valori: Vero o Falso.

bool is_game_over = false;

Il Modulo (Il Resto)

In informatica, il simbolo % non calcola la percentuale. Calcola il Resto di una divisione intera.

10 % 3 1

10 fette divise per 3 persone = 3 fette a testa. Avanza 1 fetta.

Far parlare il Computer

std::cout

Character OUTput

È il megafono del computer. Serve per stampare testi o valori delle variabili sullo schermo.

std::cout << "Ciao Mondo!";

Le frecce << spingono le parole verso lo schermo (cout).

std::cin

Character INput

È l'orecchio del computer. Blocca il programma e aspetta che l'utente scriva qualcosa sulla tastiera, salvandolo in una scatola.

std::cin >> eta_utente;

Le frecce >> spingono il dato dalla tastiera (cin) dentro la variabile.
