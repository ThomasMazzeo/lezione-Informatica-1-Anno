Introduzione al C++

Le Basi del Codice

"Nessuna formula magica.
Nessun concetto dato per scontato.
Smontiamo il codice riga per riga e impariamo a parlare con la macchina."

Usa le frecce per scorrere
Step 1

Lo Scheletro (Il Motore)

motore.cpp
// Tutto inizia da qui:
int main() {

// ... il tuo codice ...

return 0;
}

main() (La Stanza Principale)

Il computer non legge a caso. Cerca sempre la parola main e inizia a eseguire le istruzioni da lì. È il motore del programma.

{ } Le Parentesi Graffe

Rappresentano i muri della stanza. Tutto ciò che sta tra { (inizio) e } (fine) appartiene al main.

return 0;

È il segnale di "Missione Compiuta". Dice al computer: "Ho finito tutto e ci sono zero errori".

Step 2

Parlare allo Schermo

stampa.cpp
#include <iostream>

int main() {
std::cout << "Ciao Mondo!";
}
#include

Il Dizionario

Il PC nasce senza parole. <iostream> gli insegna i comandi base per l'Input e l'Output (lo schermo e la tastiera).

cout <<

Il Megafono

C-Out (Character Output). Le doppie frecce << indicano la direzione: stiamo spingendo il testo verso lo schermo.

;

Il Punto e Virgola

REGOLA D'ORO: Ogni istruzione in C++ deve finire con il punto e virgola. Dimenticarlo = Esplosione.

Step 3

Le Variabili in Sintassi

La Memoria

Per creare una "scatola" nel codice servono 3 cose: La Forma, L'Etichetta e l'Assegnazione (=).

int vite = 3;
  • int La Forma: Solo numeri Interi (niente virgola).
  • vite L'Etichetta: Il nome che abbiamo inventato noi.
  • = 3 Prendi il 3 e sbattilo dentro la scatola.
